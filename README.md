PingOne Davinci Widget Example Code

Front-End: React
Back-End: Node.js w/Fastify

Runs in Docker container

** IMPORTANT NOTE **

You will need to set up a local domain in your /etc/hosts file and allow this as an origin in the PingOne Davinci Application in the Admin Console.

Documentation:
https://docs.pingidentity.com/davinci/integrating_flows_into_applications/davinci_launching_a_flow_with_the_widget.html
